안정원(Jungwon An)
================

힘든 일 속에서도 항상 즐거움을 찾으며 함께하는 사람들과 다 함께 걸어가기 위해 자신의 분야뿐만 아니라 새로운 길이더라도 이해하기 위해 공부하고 노력하는 개발자이자, 같이 놀고 즐기는 것을 좋아하여 힘든 일 또한 발전할 수 있는 좋은 경험이라는 생각으로 빠르게 털고 일어나는 힘이 있는 사람이다.
